RealtimeIRL Twitch Extension

A simple Twitch Extension to share your location using RealtimeIRL.

Install now!

How to use this extension?

Prerequisites

  • Create a RealtimeIRL account.
  • Retrieve your "Push Key", which is displayed when you click on "Configure".
  • Download the RealtimeIRL mobile application (Android, iOS)
  • Open the mobile application, tap on "RealtimeIRL" title, tap on "RealtimeIRL push key" and enter your "Push Key".

Extension Installation

  • Install the extension
  • Select the component in which the extension will be shown.

Sharing your location

  • From the RealtimeIRL application, press the "Play" button, which will start logging your location. Once this is done your viewers will be able to see your location using the extension (your location will also be visible from the RealtimeIRL website)
